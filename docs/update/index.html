CilView.application

This download is a ClickOnce manifest link. Use it if you want to recieve CIL View updates automatically. The installer will create desktop shortcut for the application. When you run it, ClickOnce will automatically check for new version and download one if it's found. On the next application run ClickOnce will show you a dialog box offering to install the new version.

Using auto-update requires stable internet connection and access to the https://msdn-whiteknight.github.io/ website. If you are using old Windows or .NET Framework versions, you might be unable to connect due to TLS protocol version or chiphersuite mismatch. Consider downloading CIL View via Github releases if you have issues with ClickOnce installer.
